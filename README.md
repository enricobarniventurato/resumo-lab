# resumo-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO

## O que aprendi
Durante o laboratório, aprendi como acessar o portal do Microsoft Azure e realizar algumas configurações de personalização. Foi demonstrado como alterar o idioma da plataforma, modificar o tema e ajustar o layout da interface de acordo com as preferências do usuário.

Também foram apresentados os inúmeros serviços disponibilizados pelo Microsoft Azure, abrangendo diferentes áreas da computação em nuvem. Além disso, foram discutidos os riscos de desenvolver aplicações utilizando serviços experimentais ou que ainda estejam em versão de prévia.

Esses serviços podem sofrer alterações significativas ou até mesmo ser descontinuados antes de sua implementação definitiva. Caso uma aplicação dependa diretamente deles, existe o risco de ocorrerem falhas, perda de funcionalidades ou até o comprometimento de todo o projeto, inclusive em ambiente de produção.

Por esse motivo, é importante avaliar cuidadosamente o uso de tecnologias experimentais. A Microsoft informa previamente que esses serviços podem apresentar instabilidades, mudanças ou descontinuação, não garantindo o mesmo nível de suporte e confiabilidade oferecido aos serviços oficialmente disponíveis.
